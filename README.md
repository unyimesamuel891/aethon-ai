# Duncan Studio

> AI generation tools built for studios that care about the craft.

**[Live Site →](https://duncan-studio-kohl.vercel.app/)**

---

## What It Is

Duncan Studio is a landing page for an AI-powered creative platform that helps studios, small teams, and independent creators generate images, text, music, and video — without the bloat of a typical SaaS dashboard.

The UI is designed to feel like a workspace, not industrial equipment.

---

## Sections

- **Hero** — Bold editorial headline with live stats (2,400 projects, 180K creators)
- **Tools** — Five core generation tools: Image, Text, Music, Collaborate, Marketplace
- **Studio Walkthrough** — Step-by-step breakdown of the generation workflow with a live UI mockup (prompt → diffusion steps → export)
- **Case Studies** — Three real-world client stories (ceramics studio in Berlin, fashion house in Lagos, games studio in London)
- **Pricing** — Three honest tiers: Explorer (free), Creator ($24/mo), Studio ($89/mo)
- **Footer** — Full navigation, social links, sign-up modal

---

## Tech Stack

- **HTML5 & CSS3** — Semantic markup, custom layout
- **Vanilla JavaScript** — Modal interactions, smooth scroll, UI behavior
- **Vercel** — Deployment

---

## Design Highlights

- Editorial-first typography and copywriting
- Dark, minimal aesthetic with intentional whitespace
- Mobile responsive layout
- No frameworks — pure HTML/CSS/JS
